Alert

Alerts come in four roles: success, danger, warning, and info. They can be dismissible, and have a title, description, and action.

Success

Alert heading

Aww yeah, you successfully read this important alert message. This example text is going to run a bit longer so that you can see how spacing within an alert works with this kind of content.

  <div class="flex">
    <div class="flex-shrink-0">
      <i class="fa fa-check-circle text-xl text-green-500"></i>
    </div>
    <div class="ml-3">
      <h3 class="text-sm font-medium text-green-800">Alert heading</h3>
      <div class="mt-2 text-sm text-green-700">
        <p>Aww yeah, you successfully read this important alert message. This example text is going to run a bit longer so that you can see how spacing within an alert works with this kind of content.</p>
      </div>
      <div class="mt-4">
        <div class="-mx-2 -my-1.5 flex">
          <button type="button" class="bg-green-50 px-2 py-1.5 text-sm font-medium text-green-800 hover:bg-green-100 focus:outline-none focus:ring-2 focus:ring-green-600 focus:ring-offset-2 focus:ring-offset-green-50">Button text</button>
        </div>
      </div>

          <span class="sr-only">Dismiss</span>
          <i class="far fa-times w-4"></i>
        </button>
      </div>
    </div>
  </div>
</div>
